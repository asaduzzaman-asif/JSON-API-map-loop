# JSON API Map Loop

A JavaScript project that fetches user data from a JSON API and dynamically displays it in a responsive HTML table using the `fetch()` API and the `map()` method.

## 📋 How It Works

1. The application sends a request to a public JSON API.
2. The API returns user data in JSON format.
3. JavaScript fetches the data using the Fetch API.
4. The `map()` method loops through the data.
5. Each user's information is inserted into the HTML table dynamically.
